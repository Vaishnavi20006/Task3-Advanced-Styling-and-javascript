Overview

A responsive web project built using HTML, CSS, and JavaScript. It adapts to mobile, tablet, and desktop screens, includes an interactive quiz, and fetches real-time data using a public API.

Technologies

HTML5

CSS3 (Media Queries)

JavaScript

Public API

Features

Responsive design

Interactive JavaScript quiz

Dynamic API data display

How to Run

Open index.html in a browser or use Live Server in VS Code.
